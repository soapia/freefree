  • 2.

    Create an automation in Apple Shortcuts to run this shortcut daily:

    1. Click the Automation tab at the bottom of your screen, and click the button to create a new automation.
    2. Create a new personal automation.
    3. Select the trigger "Time of Day" and set it to run daily.
      Optionally, select "Run Immediately" in order to send the emails without confirming each day. This will truly automate the emails.
    4. Add the action "Run Shortcut" and choose the shortcut you downloaded.
    5. Save the automation, and you're good to go!
